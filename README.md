# 🎯 Katia Color Picker

A fast, modern desktop color picker built in Python.

---

## ✨ Features

- 🔍 Pixel-perfect color picking
- 🎯 Zoom lens (scroll to zoom)
- 🎨 Palette history
- 📋 Copy HEX & RGB instantly
- 🎚 RGB sliders + manual input
- 📦 Export palette (PNG + CSS)
- 🌙 Dark mode UI

---

## 🚀 Download

👉 https://github.com/shellatol-commits/katia-color-picker/releases/latest

---

## 🎮 Controls

- CTRL + SHIFT + C → Start picker
- Scroll → Zoom in/out
- Click → Select color
